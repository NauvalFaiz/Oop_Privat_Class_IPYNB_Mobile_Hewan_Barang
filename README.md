## C. Analisis Konsep OOP

### 1. Class dan Object

**Class** digunakan sebagai cetakan untuk membuat karakter game, sedangkan  
**Object** merupakan hasil instansiasi dari class tersebut.

Contoh:

```python
hero1 = Mage("Eudora", 100, 20)
monster = Monster("Orc", 150, 10)
```
